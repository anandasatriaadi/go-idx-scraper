# idx-go-fetch

idx-go-fetch is a Go-based tool for downloading stock data from the Indonesia Stock Exchange (IDX). Currently this is a ported from my personal project written in Python.

## Features

- Download stock data from IDX
- Support for multiple stock symbols
- Easy-to-use command-line interface
- Configurable date ranges for data retrieval
